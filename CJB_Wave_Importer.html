CJB Comfort — Wave Customer Importer

Imports customers and their properties from a Wave CSV export into Airtable. Skips anyone already in Airtable by email. Safe to run multiple times.

⚠ Export from Wave first: Wave → Customers → Export CSV. Save the file and upload it below.
ℹ What this tool does:
For each row in the CSV:
1. Checks if a Customer with that email already exists in Airtable — skips if yes.
2. Creates the Customer record (name, first/last name, email, phone, type = Residential).
3. If the row has a service address, checks if a Property already exists for that email — skips if yes.
4. Creates the Property record linked to the new Customer (address, city, state, zip).
Rows with no email are imported by name match only.
